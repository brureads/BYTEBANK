# 💙 BYTEBANK

Projeto desenvolvido durante a **Formação TypeScript da Alura**, com o objetivo de praticar os conceitos iniciais de desenvolvimento web utilizando **HTML**, **CSS**, **JavaScript** e **TypeScript**.

## Sobre o projeto

O **BYTEBANK** é um projeto inspirado em um sistema bancário simples, criado para treinar a estruturação de páginas web e a aplicação de lógica com TypeScript.

Durante o desenvolvimento, são trabalhados conceitos importantes como:

- Organização de arquivos em um projeto web;
- Estruturação de páginas com HTML;
- Estilização com CSS;
- Primeiros passos com TypeScript;
- Manipulação de dados e regras de negócio;
- Boas práticas de versionamento com Git e GitHub.
